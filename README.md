# Última Limpieza
¡Bienvenid@ a la **Última Limpieza**! Nosotros somos Aleix y Aitana el cual presentamos el juego retro de acción y exploración donde eres un "Data Sweeper" (Barredor de Datos) dentro de un servidor corrupto por un virus masivo. Tu misión es recuperar 3 "Paquetes de Datos Vitales" antes de que el sistema se reinicie por completo.
